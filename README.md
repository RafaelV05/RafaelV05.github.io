Professional Self-Assessment

Throughout the Computer Science program at Southern New Hampshire University, I have developed both technical and professional skills that have prepared me for a career in software development. The coursework provided experience in software engineering, algorithms and data structures, databases, security, testing, and full-stack development. As I progressed through the program, I became more confident in my ability to design, develop, and improve software solutions while applying industry best practices.

The artifact selected for this ePortfolio is the Travlr Getaways web application, originally developed in CS-465 Full Stack Development. I chose this project because it demonstrates a wide range of skills developed throughout the program and provided opportunities for meaningful enhancements in software design and engineering, algorithms and data structures, and databases. By enhancing a single artifact across multiple categories, I was able to demonstrate how different areas of computer science work together to improve a real-world application.

One of the most valuable skills I developed during the program was communication. Whether working through project requirements, documenting software, or presenting technical information, I learned how to communicate complex ideas clearly and effectively. The narratives included in this portfolio demonstrate my ability to explain technical decisions, justify design choices, and reflect on the development process.

Collaboration was another important area of growth. While many projects were completed individually, I gained experience working within development practices that mirror professional team environments. Using Git and version control, following software development lifecycles, and organizing code into maintainable components helped me understand how software projects are managed and maintained in collaborative settings.

The software design and engineering enhancement focused on improving maintainability, security, and organization through middleware-based authentication and authorization. This enhancement strengthened my understanding of software architecture and demonstrated my ability to design modular and reusable solutions.

The algorithms and data structures enhancement focused on improving how application data is processed and presented to users. By implementing searching, filtering, and sorting functionality, I demonstrated how data structures and algorithms can improve efficiency and user experience.

The database enhancement focused on improving data integrity, validation, and performance. Adding schema validation, indexes, and structured data constraints strengthened the reliability of the application and demonstrated my understanding of database design principles.

Security has been a recurring theme throughout my coursework and is reflected throughout this project. Authentication, authorization, validation, and secure API design were incorporated into the enhancements to help protect application data and ensure only authorized users could perform specific actions.

Together, these enhancements transformed the Travlr Getaways application into a more secure, maintainable, and scalable system. More importantly, they demonstrate the technical knowledge and problem-solving skills I developed throughout the Computer Science program. This portfolio represents not only the work I completed during my degree but also my growth as a developer and my readiness to continue learning and contributing within the software development field.
