## Artifact Overview

The artifact selected for this capstone project is Travlr Getaways, a full-stack travel booking application originally developed in CS-465 Full Stack Development.

## Technologies Used

- Node.js
- Express
- Angular
- MongoDB
- Mongoose
- JWT Authentication

## Reason for Selection

I selected this artifact because it provided opportunities to demonstrate skills in software engineering, algorithms and data structures, and database development through a single application.
